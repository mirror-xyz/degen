## Install

```bash
yarn add degen
```

## Set Up

Wrap the root of your app in a [`ThemeProvider`](/components/ThemeProvider), import the external stylesheet, and add [Inter](https://rsms.me/inter/) and [iA Writer Mono](https://github.com/iaolo/iA-Fonts/tree/master/iA%20Writer%20Mono) to your font stack.

```tsx
import { ThemeProvider } from 'degen'
import 'degen/styles'

const App = () => {
  return <ThemeProvider>{children}</ThemeProvider>
}
```

import { Button, IconLockClosed, Stack } from 'degen'
