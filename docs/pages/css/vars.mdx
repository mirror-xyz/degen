## Usage

You can use `vars` within [vanilla-extract](https://vanilla-extract.style/) stylesheets (see above) and runtime code. You should only use `vars` if youâ€™re unable to use [`Box`](/components/Box) or [`atoms`](/css/atoms).

```tsx
import { vars } from 'degen'

const FancyButton = () => {
  return (
    <button
      style={{
        backgroundColor: vars.colors.background,
        fontSize: vars.fontSizes.small,
        padding: vars.space['4'],
      }}
    >
      Click Me
    </button>
  )
}
```
