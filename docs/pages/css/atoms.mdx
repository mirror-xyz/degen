## Usage

You can use `atoms` within [vanilla-extract](https://vanilla-extract.style/) stylesheets (see above). These utilities are recommended where possible to reduce the amount of custom CSS in your application.

export default ({ children }) => <Layout meta={meta}>{children}</Layout>
