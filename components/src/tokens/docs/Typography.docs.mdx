## Fonts

```tsx live=true
<Stack>
  <Text font="sans">Sans</Text>
  <Text font="mono">Mono</Text>
</Stack>
```

## Font Sizes

```tsx live=true
<Stack>
  <Heading level="1">Heading One</Heading>
  <Heading level="2">Heading Two</Heading>
  <Heading level="3">Heading Three</Heading>
  <Text variant="extraLarge">Extra Large</Text>
  <Text variant="large">Large</Text>
  <Text variant="base">Base</Text>
  <Text variant="small">Small</Text>
  <Text variant="label">Label</Text>
</Stack>
```

## Letter Spacing

```tsx live=true
<Stack>
  <Text letterSpacing="-0.02">Spacing (-0.02)</Text>
  <Text letterSpacing="-0.015">Spacing (-0.015)</Text>
  <Text letterSpacing="-0.01">Spacing (-0.01)</Text>
  <Text letterSpacing="normal">Spacing (normal)</Text>
  <Text letterSpacing="0.03">Spacing (0.03)</Text>
</Stack>
```
