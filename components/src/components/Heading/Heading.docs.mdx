---
title: Heading
---

Page headings and titles.

```tsx
import { Heading } from 'degen'
```

```tsx live=true
<Heading>Hello World</Heading>
```

## Levels

By default, the semantic heading level matches the visual heading level, e.g. `<Heading level="2">` will render an `h2` element. If you need the semantics to be different from the visuals, you can provide the desired HTML tag via the `as` prop.

```tsx live=true
<Stack direction="vertical">
  <Heading level="1">Level 1</Heading>
  <Heading level="2">Level 2</Heading>
  <Heading as="h3" level="2">
    As h3
  </Heading>
</Stack>
```

## Align

```tsx live=true
<Stack direction="vertical">
  <Heading align="left">Left</Heading>
  <Heading align="center">Center</Heading>
  <Heading align="right">Right</Heading>
</Stack>
```

## Color

```tsx live=true
<Stack direction="vertical">
  <Heading color="foreground">Foreground</Heading>
  <Heading color="text">Text</Heading>
  <Heading color="textSecondary">Text Secondary</Heading>
</Stack>
```
