## Content Types

Uses HTML [`accept`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/accept) attribute.

```tsx live=true
<Stack>
  <MediaInput
    accept="image/*"
    compact
    label="Choose or drag and drop an image"
  />
  <MediaInput
    accept="video/*"
    compact
    label="Choose or drag and drop an video"
  />
</Stack>
```
