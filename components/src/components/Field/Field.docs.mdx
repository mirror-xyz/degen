## Consume ID

The content wrapped by field sets an `id` automatically for accessibility. You can customize the behavior by using the render prop.

```tsx live=true
<Stack>
  <Field label="Automatically injected">
    <Box backgroundColor="foregroundSecondary" height="32px" />
  </Field>

  <Field label="Render prop">
    {(ids) => (
      <Box
        backgroundColor="foregroundSecondary"
        height="32px"
        {...ids.content}
      />
    )}
  </Field>
</Stack>
```
