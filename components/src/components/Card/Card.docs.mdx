<PropsTable sourceLink={sourceLink} types={types} />

## Shadow

Adds shadow when `mode` is `light`.

```tsx live=true expand=true
<Card padding="6" shadow>
  <Heading>Hello World</Heading>
  <Text>The quick brown fox…</Text>
</Card>
```

## Hover

Adds an outline on hover.

```tsx live=true expand=true backgroundColor=foregroundTertiary
<Card hover padding="6">
  <Heading>Hello World</Heading>
  <Text>The quick brown fox…</Text>
</Card>
```
