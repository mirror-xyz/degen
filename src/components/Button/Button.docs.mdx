# Button

Used to trigger an operation.

## Size

```tsx live=true
<Stack align="center">
  <Button size="lg">Connect Wallet</Button>
  <Button size="md">Connect Wallet</Button>
</Stack>
```

## Tone

```tsx live=true
<Stack align="center">
  <Button tone="accent">Connect Wallet</Button>
  <Button tone="blue">Connect Wallet</Button>
  <Button tone="green">Connect Wallet</Button>
  <Button tone="red">Connect Wallet</Button>
</Stack>
```

## Variant

```tsx live=true
<Stack align="center">
  <Button variant="highlight">Connect Wallet</Button>

  <Button variant="secondary">Connect Wallet</Button>
  <Button variant="tertiary">Connect Wallet</Button>

</Stack>
```

## Loading

